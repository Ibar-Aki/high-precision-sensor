# 高精度傾斜角センサー PWA 精度推定レポート
更新日: 2026-02-13

## 1. 目的

本レポートは、実装仕様およびモバイル端末特性に基づき、本アプリケーションの期待精度を推定した結果を記録するものである。

## 2. 推定結果（要約）

| 評価項目 | 推定値 | 補足 |
|------|------|------|
| 分解能 | 0.01° 〜 0.05° | センサー量子化および API 提供粒度の影響を含む |
| 静的安定性 | ±0.05° 程度 | フィルタ連鎖により微小揺らぎを抑制 |
| 絶対精度 | ±1.0° 〜 3.0° | 端末個体差、設置姿勢、機構誤差の影響を受ける |
| 相対精度 | ±0.1° 〜 0.2° | キャリブレーション実施時の変化量計測として想定 |

## 3. 推定根拠

### 3.1 センサーおよび API 起因の制約

- DeviceOrientation API の提供値はブラウザ実装に依存する。
- プライバシー対策に伴う量子化や丸め処理が精度に影響する。
- 端末搭載センサーにはホワイトノイズおよび温度依存ドリフトが存在する。

### 3.2 アルゴリズムによる補正効果

- 1D カルマンフィルタ: 外乱性の高い観測値を確率的に抑制。
- EMA: 高周波成分を平滑化し、表示の過敏な変動を軽減。
- デッドゾーン: 視認上の不要な微小変動を遮断。

## 4. 高精度運用条件

1. 測定前に必ずキャリブレーションを実施すること。
2. 端末支持面の機械的平面性を確保すること。
3. 温度変化後は再キャリブレーションを実施すること。
4. 絶対値評価と相対変化評価を用途別に区別すること。

## 5. 総合評価

本アプリケーションは、絶対精度を要求する計量機器の代替を目的とするものではない。一方、設備調整における相対傾斜の把握という運用目的においては、実用に耐える安定性と追従性を有すると判断する。

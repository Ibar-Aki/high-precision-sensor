高精度傾斜角センサー PWA ─ プロジェクト概要

2026.02

開発動機：現場の水平出し

  • 1. 既存計測手段の限界：デジタルプロトラクターは数万円〜で全員配布は非現実的。気泡管は目視読取に個人差があり±0.5°が限界。一方、全員のポケットに高性能MEMSセンサー搭載のスマホがある。
  • 2. 作業中に画面が見えない：水平出し作業は両手がふさがる。数値を確認するたびに手を止める必要があり、作業効率が大幅に低下する。→ 音声フィードバックで解決。

解決策：中核機能

■ カルマンフィルタ＋EMAによる高精度計測

3段フィルタ連鎖でセンサーノイズを確率的に除去。分解能0.01°を実現。

■ 音声ガイドでハンズフリー計測

Web Audio APIによる4方向フィードバック音。画面を見ずに傾きを把握可能。

■ PWA＝インストール不要で即利用

ブラウザで開くだけで動作。オフライン対応。ホーム画面に追加でネイティブ感覚。

動作環境・利用条件

  • ● 対応端末：iPhone / Android / PC のブラウザ（PWA対応）
  • ● 通信環境：完全オフライン動作。ネット接続不要
  • ● データ保管：端末ローカルのみ（外部送信なし）
  • ● ライセンス：MIT License（オープンソース）

計測フロー

1. センサー有効化

DeviceOrientation APIの許可をリクエスト。iOSではユーザー操作が必須。

▼

2. キャリブレーション

1点校正（現在値を0にリセット）または2点校正（180度反転法でオフセット除去）。

▼

3. フィルタ処理パイプライン

Kalman → EMA → Deadzone の3段フィルタ。静止時はStatic Average自動切替。

▼

4. リアルタイム表示・音声出力

SVG水準器＋デジタル数値表示。方向別音声ガイド or 角度読み上げ。

プレミアムUI＆UX

UI

ダークモード＋グラスモーフィズム

OLED映えする高級感あるデザイン。現場での視認性も高い。

UI

SVGリアルタイム水準器

アナログ水準器を模した直感的なビジュアル。バブルの動きで傾きを即座に把握。

UX

4方向音声フィードバック

前後左右の傾き方向と量を音の高さ・パターンで通知。ハンズフリーで作業可能。

UX

2点キャリブレーション

180度反転法でセンサー固有オフセットを除去。より正確な絶対値計測を実現。

生成AI × 現場アプリ ── 3つのモックアップ紹介

生成AI技術の可能性を示すプロトタイプ集

2026.02 社内向け
紹介の目的
①

アイデアを膨らませる。簡易モックアップを出発点に、「こうしたら面白い」「こっちにも使える」を自由に発想したい。

②

生成AIの実力を把握する。音声認識・対話AI・構造化抽出など、既に実用水準。1回あたり約0.1円の低コストで動く実例として紹介。

③

依頼研究・外注の構想材料に。「ここまでは個人で試作できた」を起点に、本格開発の依頼先選定や共同研究の検討を始めたい。

01
SENSOR × PWA

高精度傾斜角センサー

スマホ内蔵の加速度センサーをカルマンフィルタで高精度化。分解能0.01°でリアルタイム計測。音声フィードバックでハンズフリー計測が可能。

  • 3段フィルタ連鎖（Kalman → EMA → Deadzone）
  • 傾き方向を音の高さ・パターンで通知
  • PWA＝インストール不要、完全オフライン動作

使用技術

DeviceOrientation Kalman Filter Web Audio PWA
02
SENSOR × GUIDE

机の水平ガイド

机の傾きを計測し、4隅のボルト回転量をmm→回転数で自動算出。音声ガイドで読み上げ、ボルトを回しながら耳で確認できる。

  • 三角関数で各脚の高さ差→回転量変換
  • 静止判定→自動確定（ボタン操作不要）
  • 日本語/英語 自動切替・完全オフライン

使用技術

DeviceOrientation SpeechSynthesis Kalman Filter PWA
03
AI × VOICE

Voice KY Assistant

AIがファシリテーターとして危険予知（KY）活動を支援。音声対話で深掘り質問→構造化→KYシートPDFを自動生成。一人KYの形骸化を防止。

  • AIが深掘り質問 → 新たな視点をサジェスト
  • 音声入力でハンズフリー記録（手袋OK）
  • 監査対応PDF即時出力（二度書き排除）

使用技術

Gemini 2.5 Web Speech React 19 Cloudflare Workers

▍ これらの試作で分かったこと

  • スマホ標準機能だけで専用計測器に近い精度を実現可能
  • AI対話コスト：1回KYあたり約0.1円（Gemini利用時）
  • 音声UI＋PWAの組合せで「現場で使える」レベルに到達

▍ 開発体制・期間

  • 開発者1名＋生成AIコーディングアシスタント
  • 各アプリ約1〜3週間で試作完了
  • 全てWebブラウザで動作、専用サーバー不要（03除く）

▍ 今後のステップ案

  • 現場ニーズの深掘り → 優先テーマの選定
  • 外部パートナーとの共同開発・PoC実証

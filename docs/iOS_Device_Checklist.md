# iOS実機確認チェックリスト
作成日時: 2026-02-14 01:54:13 +09:00
作成者: Codex＋GPT-5

## 1. 事前条件
- [ ] iPhone がインターネット接続可能である。
- [ ] 公開 URL が HTTPS で配信されている。
- [ ] Safari の「モーションと画面の向きのアクセス」が有効である。

## 2. 権限フロー確認
- [ ] 初回起動時に「センサーを有効にする」押下で権限ダイアログが表示される。
- [ ] 許可時に計測画面へ遷移する。
- [ ] 拒否時にエラーメッセージが表示される。
- [ ] 拒否後に再試行手順が利用者に理解できる文言で示される。

## 3. センサー追従性確認
- [ ] 前後方向に端末を傾けた際、ピッチ値と方向表示が連続更新される。
- [ ] 左右方向に端末を傾けた際、ロール値と方向表示が連続更新される。
- [ ] 静止時に数値が過度に振動しない。
- [ ] キャリブレーション後、基準姿勢で 0 付近に収束する。
- [ ] 値ロック機能が有効/無効で意図通り動作する。

## 4. 音声機能確認
- [ ] 音声 ON/OFF が切替可能である。
- [ ] 前後左右で音色差が認識できる。
- [ ] 閾値モードで設定角度未満は無音、超過時は音が出る。
- [ ] 音量スライダー変更が反映される。

## 5. PWA インストール確認
- [ ] Safari の共有メニューから「ホーム画面に追加」が実行可能である。
- [ ] 追加後、ホーム画面アイコンから起動できる。
- [ ] 起動時に standalone 表示で動作する。
- [ ] 一度読み込み後、通信切断時でも主要画面が表示される。

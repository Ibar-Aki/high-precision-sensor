机の水平ガイド PWA ─ プロジェクト概要

2026.02

開発動機：机の水平調整

  • 1. 調整作業が非効率：4隅のボルト回転量が分からず、水準器を見ながら手探りで調整。何度もやり直しが発生し、1台15〜30分かかることも。
  • 2. 作業中に画面が見えない：ボルトを回す両手がふさがり、数値を確認するたびに手を止める。→ 音声ガイドで解決。

解決策：中核機能

■ 4隅ボルト回転量を自動算出

傾きから各脚の調整量をmm単位で計算。ボルトピッチから回転数に変換して表示。

■ 音声ガイドでハンズフリー

調整方向と回転量を音声で読み上げ。ボルトを回しながら耳で確認できる。

■ 静止判定→自動確定

スマホを机に置くだけで自動的に安定検出。ボタン操作なしで計測完了。

動作環境・利用条件

  • ● 対応端末：iPhone / Android / PC のブラウザ（PWA対応）
  • ● 通信環境：完全オフライン動作。ネット接続不要
  • ● データ保管：端末ローカルのみ（外部送信なし）
  • ● 多言語対応：日本語 / 英語 自動切替

計測フロー

1. センサー有効化

DeviceOrientation APIをリクエスト。

▼

2. 机にスマホを設置

計測開始ボタンを押して机上に置く。

▼

3. 自動静止判定 → 自動確定

分散監視で安定を検出。MEASURING到達で自動確定。タイムアウト時は手動確定も可。

▼

4. 音声ガイド＋調整量表示

4隅のボルト回転量を表示＆読み上げ。1本調整ごとに再計測。

音声ガイド機能

🔊

全脚が許容範囲内に収まると「水平達成」を読み上げ。画面を見ずに完了を確認。
